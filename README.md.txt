# 🛒 Superstore Sales Analysis (Excel)

This project analyzes sales data from a fictional Superstore to extract business insights using Microsoft Excel. It includes data cleaning, pivot table summaries, visual dashboards, and KPIs to guide decision-making.

## 📁 Project Files

- `Superstore_Analysis.xlsx` – Excel workbook with data, pivot tables, charts, and a dynamic dashboard.
- `README.md` – This project documentation file.

## 📊 Key Features

- Organized and cleaned raw sales data
- Pivot tables for segmenting data by region, category, and time
- Charts showing sales and profit trends
- Interactive dashboard with slicers and KPIs

## 🔍 Business Insights

- Top-performing regions and customer segments
- Most profitable product categories
- Monthly/yearly sales trends
- Discounting’s effect on profit margins

## 🛠 Tools Used

- Microsoft Excel:
  - Pivot Tables
  - Charts & Conditional Formatting
  - Dashboard Design
  - Slicers & Timelines

## 📈 How to Use

1. Download or clone this repository.
2. Open `Superstore_Analysis.xlsx`.
3. Navigate to the `Dashboard` sheet.
4. Use slicers to filter insights by region, category, and more.
